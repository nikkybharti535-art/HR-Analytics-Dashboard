# HR Analytics Dashboard

## Project Overview

This project is an interactive HR Analytics Dashboard developed using Power BI. The dashboard helps analyze employee attrition and workforce trends by providing insights into employee demographics, salary distribution, education background, age groups, and job roles.

## Tools Used

- Power BI
- Excel
- Power Query
- DAX

## Dashboard Features

- Employee Count
- Attrition Count
- Attrition Rate
- Average Employee Age
- Average Salary
- Average Years at Company
- Department-wise Filtering
- Attrition Analysis by Education
- Attrition Analysis by Age Group
- Attrition Analysis by Salary
- Attrition Analysis by Job Role
- Attrition Analysis by Years at Company

## Key Insights

- The overall attrition rate is 16.1%.
- Employees in the 26–35 age group show the highest attrition.
- Lower salary ranges have comparatively higher attrition.
- Attrition varies across different job roles and education backgrounds.
